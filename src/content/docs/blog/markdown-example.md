# Markdown記事の例

これは通常のMarkdownで書かれた記事です。TypstとMarkdownを混在させることができます。

## Markdownの特徴

- 簡単に書ける
- 広く使われている
- Starlightとの相性が良い

## コード例

```javascript
console.log("Hello from Markdown!");
```

## まとめ

TypstとMarkdownを使い分けることで、柔軟なブログ運営が可能です。
