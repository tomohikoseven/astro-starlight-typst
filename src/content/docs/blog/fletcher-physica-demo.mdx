## 🛠️ インポート方法

プロジェクト内の `/libs/` ディレクトリ配下に配置されているため、`.mdx`（数式ブロック内）や `.typ` ファイルから以下のようにルート相対パスでインポートできます。

```typst
// Fletcher のインポート
#import "/libs/fletcher/src/exports.typ" as fletcher: diagram, node, edge

// Physica のインポート
#import "/libs/physica/physica.typ": *
```

## 1. Physica による洗練された物理・数学表記

まずは `physica` ライブラリの機能を見てみましょう。微分やディラック表記などの複雑な式を劇的に簡潔に記述できます。

### 偏微分と微分
通常の Typst では分数と偏微分記号（`partial`）を組み合わせて長く書く必要がありますが、`physica` では `#pdv` や `#dv` を用いてシンプルに記述できます。

$$
#import "/libs/physica/physica.typ": *

#pdv($f$, $x$, $y$) = #pdv($f$, $y$, $x$) \
#dv($y$, $x$, $2$) = dif^2 y / dif x^2
$$

### ベクトルと行列演算
ボールド体のベクトル `#vb` や、二次元回転行列 `#rot2mat` などを素早く生成できます。

$$
#import "/libs/physica/physica.typ": *

#vb($E$) = - nabla V - #pdv(vb($A$), $t$) quad text("and") quad #rot2mat($theta$) = mat(cos theta, -sin theta; sin theta, cos theta)
$$

### 量子力学：ディラックのブラケット記法
量子力学で頻出するブラ（`bra`）、ケット（`ket`）、ブラケット（`braket`）、外積（`ketbra`）、期待値（`expval`）が綺麗に定義されています。

$$
#import "/libs/physica/physica.typ": *

#braket($phi$, $hat(H)$, $psi$) = #expval($hat(H)$, $psi$) \
#ketbra($psi_i$, $psi_j$) = #ket($psi_i$) #bra($psi_j$)
$$

## 2. Fletcher による美しい図式描画

`fletcher` は、従来の CeTZ よりもさらに「可換図式」や「状態遷移図」を描画することに特化したライブラリです。数式モードと親和性の高いグリッド記法をサポートしています。

### カテゴリ論・可換図式（第一同型定理）
数式中に `&`（カラム区切り）と `\`（行区切り）を使用するだけで、綺麗に整列されたダイアグラムを描画できます。

$$
#import "/libs/fletcher/src/exports.typ" as fletcher: diagram, node, edge

#diagram(cell-size: 17mm, $
  G edge(f, ->) edge("d", pi, ->>) & im(f) \
  G slash ker(f) edge("ur", tilde(f), "hook-->")
$)
$$

### カスタム形状とスタイルを用いたフローチャート
長方形の他にも、`diamond`（ひし形）、`hexagon`（六角形）などの豊富なノード形状をサポートしており、グラデーションやカスタムストロークを適用できます。

$$
#import "/libs/fletcher/src/exports.typ" as fletcher: diagram, node, edge
#import fletcher.shapes: diamond, hexagon

#diagram(
  node-stroke: (paint: rgb("3b82f6"), thickness: 1pt),
  node-fill: rgb("eff6ff"),
  spacing: 3em,
  node((0,0), [開始], corner-radius: 4pt),
  edge((0,0), (1,0), "-|>", label: [入力]),
  node((1,0), align(center)[分岐判定], shape: diamond, fill: rgb("fef9c3"), stroke: (paint: rgb("eab308"), thickness: 1pt)),
  edge((1,0), (2,0), "-|>", label: [Yes]),
  node((2,0), [処理A], shape: hexagon, fill: rgb("dcfce7"), stroke: (paint: rgb("22c55e"), thickness: 1pt)),
  edge((1,0), (1,1), "-|>", label: [No]),
  node((1,1), [終了], corner-radius: 4pt, extrude: (0, 2), fill: rgb("fee2e2"), stroke: (paint: rgb("ef4444"), thickness: 1pt))
)
$$

## 3. Fletcher × Physica による最強の連携例

最後に、**Fletcher の構造的な図式描画力**と、**Physica の洗練された物理・数学表記**を掛け合わせた高度な連携例を紹介します。

量子力学における**状態ベクトルの時間発展と相互作用**をダイアグラム化し、状態遷移確率の行列要素（期待値・ブラケット）を重ね合わせます。

$$
#import "/libs/fletcher/src/exports.typ" as fletcher: diagram, node, edge
#import "/libs/physica/physica.typ": *

#diagram(
  cell-size: 25mm,
  node((0,0), ket($psi_i$), name: <i>, fill: rgb("eff6ff"), stroke: (paint: rgb("3b82f6"), thickness: 1.5pt)),
  node((2,0), ket($psi_f$), name: <f>, fill: rgb("f0fdf4"), stroke: (paint: rgb("22c55e"), thickness: 1.5pt)),
  
  edge(<i>, <f>, "-|>", label: expval($hat(H)$), label-pos: 0.5, bend: 30deg, stroke: (paint: rgb("2563eb"), thickness: 1.2pt)),
  
  edge(<f>, <i>, "-|>", label: braket($psi_i$, $hat(V)$, $psi_f$), label-pos: 0.5, bend: 30deg, stroke: (paint: rgb("dc2626"), thickness: 1.2pt, dash: "dashed"))
)
$$

<Aside type="tip" title="デザインのポイント">
  ノードのラベルに直接 `ket(psi_i)` や `ket(psi_f)` などの `physica` 関数を使用でき、遷移を示す矢印のラベルに `expval(hat(H))` や `braket(...)` を直接乗せてレンダリングすることができます。これらがすべて一貫したフォントと極めて高精度なベクターグラフィックス（SVG）としてページに埋め込まれます。
</Aside>
